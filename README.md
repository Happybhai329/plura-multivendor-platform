# 🚀 Plura – Multi Vendor SaaS Platform

A full-stack SaaS platform inspired by real-world business tools, built using Next.js, Prisma, and Stripe.

This project demonstrates how agencies can manage clients, build funnels, handle subscriptions, and scale operations using a multi-tenant architecture.

---

## 🔥 What I Built

- Multi-tenant SaaS system (Agency + Subaccounts)
- Website & Funnel Builder (drag & drop editor)
- Stripe Connect integration for payments
- Role-based authentication system
- Kanban-based project management system
- Custom dashboards with analytics & metrics
- Media storage and file management
- Notification system

---

## 🛠 Tech Stack

- Frontend: Next.js 14, React, Tailwind CSS
- Backend: Node.js, Prisma ORM
- Database: MySQL
- Payments: Stripe + Stripe Connect
- Runtime: Bun

---

## ⚡ Key Highlights

- Built scalable SaaS architecture
- Implemented real-world features like subscriptions & billing
- Designed modular backend structure
- Hands-on experience with Stripe APIs and multi-user systems
